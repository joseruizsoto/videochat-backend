# VideoChat Backend

Backend para aplicación de videochat con WebRTC y Socket.IO.

## Funcionalidades

- ✅ Salas de videochat
- ✅ WebRTC P2P
- ✅ Compartir pantalla
- ✅ Compartir archivos
- ✅ Mano levantada
- ✅ Control de audio

## Despliegue

Desplegado en Railway.app
